# electron-vue 博客后台 
基于vue 打包 通过electron 构建桌面应用
(nodejs 环境)
（需要后端端口的开启 https://gitee.com/chenxiaop/coser-admin  server目录就是后端数据库了 至于搭建后端 数据库-->克隆server目录文件 node server 或者nodemon server）
#安装
1全局安装electron 
cnpm install electron -g
2 克隆项目 
3进入项目目录 打开命令窗口
安装依赖包 cnpm install
运行项目 electron .
